# 部署说明

## 当前版本信息

**版本**: v1.0.0  
**提交哈希**: 查看 `git log` 获取最新提交信息

## 恢复代码到当前版本

如果需要恢复到当前稳定版本：

```bash
# 查看提交历史
git log

# 恢复到指定提交（使用最新的提交哈希）
git checkout <commit-hash>

# 或者恢复到最新提交
git reset --hard HEAD
```

## 版本标签

建议为当前版本创建标签：

```bash
git tag -a v1.0.0 -m "第一版：基础功能完成"
git push origin v1.0.0
```
