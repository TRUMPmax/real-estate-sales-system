## 推送到GitHub

### 步骤1：在GitHub创建仓库

1. 访问 https://github.com/new
2. 创建新仓库（不要初始化README）
3. 复制仓库URL

### 步骤2：连接并推送

```bash
# 添加远程仓库
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# 重命名主分支为main（如果GitHub使用main）
git branch -M main

# 推送所有代码
git push -u origin main
```

### 步骤3：验证

访问GitHub仓库页面，确认所有文件已上传。
