## 版本标签建议

建议为重要版本创建标签：

```bash
# 创建v1.0.0标签（第一版）
git tag -a v1.0.0 15d1a31 -m "Version 1.0.0: 基础功能完成"

# 创建v1.0.1标签（登录页面优化）
git tag -a v1.0.1 92c3afb -m "Version 1.0.1: 登录页面优化"

# 查看所有标签
git tag -l

# 推送到远程仓库
git push origin --tags
```
